# Tech Stack

## Frontend

- **React** — UI library
- **Vite** — build tool and dev server
- **TypeScript** — type safety across the codebase
- **Tailwind CSS** — utility-first styling
- **React Router** — client-side routing

## Backend

- **Node.js + Express** — HTTP server and API
- **TypeScript** — type safety across the codebase
- **Prisma** — type-safe ORM
- **PostgreSQL** — relational database for tickets, users, categories, and knowledge base

## Authentication

- **Database sessions** — sessions stored in PostgreSQL via Prisma; session token passed via HTTP-only cookie

## AI

- **Anthropic Claude API** — ticket classification, summarization, and reply suggestion

## Infrastructure

- **Docker** — containerized frontend, backend, and database
- **Google Cloud Platform (GCP)** — cloud deployment target
